## Linux Volume Management (Completed ✅)

This module covers hands-on practice of Linux storage management
using AWS EC2 and EBS volumes.

### Topics Covered
- What is EBS volume
- Attach vs Mount concept
- Availability Zone dependency (EC2 & EBS)
- Creating EBS in correct AZ(Availablility Zone)
- Formatting volumes using mkfs
- Mounting volumes to Linux directories
- Verifying storage using lsblk and df -h
- Real-world troubleshooting when instance not visible

### Environment
- OS: Ubuntu (EC2)
- Cloud: AWS EC2 + EBS
